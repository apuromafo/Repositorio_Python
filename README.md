# Repositorio Python: Frameworks de Auditoría de Seguridad Cibernética

## Descripción General

Este repositorio contiene scripts Python para pruebas de penetración, auditoría de seguridad y análisis de vulnerabilidades. Está organizado en carpetas temáticas que abarcan diferentes aspectos de la seguridad ofensiva y defensiva.

### Objetivo Principal
Proporcionar herramientas prácticas para profesionales de la seguridad informática, investigadores y estudiantes interesados en la auditoría ofensiva, análisis de vulnerabilidades y pruebas de penetración.

---

## Estructura del Repositorio

### Carpetas Principales

#### 1. **001_Analisis_header_eml**
Scripts para análisis de cabeceras de correo electrónico y encabezados HTTP

#### 2. **002_Download**
Herramientas para la descarga segura de recursos

#### 3. **003_retos_Script_python**
Retos de programación orientados a la seguridad (POA)

#### 4. **004_juego_adivinanza**
Juego educativo de adivinanza (POA)

#### 5. **005_extrae_correo**
Herramientas para procesamiento y extracción de correos electrónicos

#### 6. **006_Móvil**
Herramientas para pruebas de seguridad de aplicaciones móviles

#### 7. **007_conversor a  md**
Conversores de Markdown y utilidades para procesado de texto

#### 8. **008_Cabeceras_Seguridad**
Herramientas para aplicación de cabeceras de seguridad web

#### 9. **009_Rut_Chileno**
Herramientas de validación de RUT chileno y control de acceso

#### 10. **010_malapi_json**
Herramientas para análisis de archivos JSON maliciosos
