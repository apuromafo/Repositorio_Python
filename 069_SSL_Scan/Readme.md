# SCAN SSL - Versión 34.0

Herramienta de auditoría de seguridad SSL/TLS con enfoque en cumplimiento técnico (PCI DSS, NIST) y preparación para criptografía post-cuántica (PQC).

## Requisitos
* Python 3.x
* Conexión a internet (solo para la primera ejecución, para descargar el binario `sslscan`)
* Dependencias de Python: `colorama`

Instalación de dependencias:
```bash
pip install colorama
